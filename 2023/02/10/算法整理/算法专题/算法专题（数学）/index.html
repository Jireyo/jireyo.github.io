数论

最大公约数

欧几里得算法

正整数a,b的最大公约数记为gcd(a,b)，有定理：

  • 当b = 0时，gcd(a,b) = a；
  • 当b != 0时，gcd(a,b) = gcd(b, a mod b)。
  • 如果gcd(a,b) = 1，则a,b互质。

裴蜀定理

如果gcd(a,b) = c，则对任意x和y，ax+by一定是c的倍数，且一定存在ax+by=c。
推论：若a,b互质，存在x,y使则ax+by=c。

例题

204. 计数质数

  • 埃式筛：每碰到一个质数，把这个质数的所有倍数都进行标记，最后统计所有质数。（对于a × b = x，只需要遍历将a遍历到根号x即可。）
  • 线性筛，主要为了解决埃式筛重复标记的问题。将x与已知质数a做乘积标记，当遇到x%a == 0时终止。
    • 完整性：任何非质数都可以由质数与另一个数相乘得到。
    • 剪枝：若 x%a == 0，则 x*a = (x/a) * (a*a) ，此时剪枝，因为对于任意质数b > a，肯定有x*b = (x*b/a)*a，即该值会被x*b/a遍历标记，所以此时可先结束标记。

149. 直线上最多的点数

证明如下：
a可以表示成a = kb + r；
假设d是a,b的一个公约数，记作d|a,d|b，即a和b都可以被d整除。
而r = a - kb，两边同时除以d，r/d=a/d-kb/d，由等式右边可知m=r/d为整数，因此d|r 因此d也是b,a mod b的最大公约数。

  2. 根据以上操作，可通过set去重，发现重复即出现死循环。当然数学上可以证明只有K是2、5的倍数时才会出现死循环。
